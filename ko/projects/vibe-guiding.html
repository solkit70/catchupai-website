Vibe Guiding

시스템 스스로 학습하고, 사용자를 능동적으로 안내하는 차세대 AI 가이드 시스템. 현재 구상 단계입니다.

Vibe Guiding이란?

Vibe Guiding은 하나의 시스템을 만든 후, 해당 폴더를 다른 시스템 안에 넣고 실행하면 그 시스템을 자동으로 공부하여 사용자에게 안내할 준비를 하는 AI 가이드 시스템입니다.

내부적으로 VibeLearn AI를 활용해 대상 시스템을 스스로 분석하고, 사용자가 시스템을 사용하는 동안 질문하거나 도움이 필요할 때 능동적으로 가이드를 제공합니다.

핵심 동작 원리

  • 자동 시스템 학습: 지정된 폴더를 분석하여 해당 시스템의 구조와 기능을 VibeLearn AI로 자동 학습합니다.
  • 가이드 준비 단계: 학습 완료 후, 사용자 질문에 답변하고 문맥에 맞는 도움말을 제공할 수 있도록 준비합니다.
  • 능동적 안내: 사용자가 질문할 때뿐만 아니라, 도움이 필요할 것으로 판단될 때 선제적으로 가이드를 제시합니다.

현재 상태

Vibe Guiding은 현재 구상 단계로, 아직 개발이 시작되지 않았습니다.

Vibe Guiding의 첫 번째 적용 대상은 Vibe Learning으로 계획되어 있습니다. 때문에 현재는 Vibe Learning 개발에 집중하고 있으며, Vibe Guiding은 이후 기회가 될 때 본격적으로 착수할 예정입니다.

왜 Vibe Guiding인가?

복잡한 시스템일수록 사용자가 처음 접할 때 어려움을 겪습니다. Vibe Guiding은 매뉴얼을 읽는 대신, AI가 직접 시스템을 이해하고 사용자 곁에서 실시간으로 도움을 주는 방식으로 이 문제를 해결하려 합니다.
