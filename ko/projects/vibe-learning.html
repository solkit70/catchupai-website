Vibe Learning

강요된 학습이 아닌 흐름에 따른 학습 — 50회의 Vibe Coding에서 탄생한 방법론으로, AI 보조 개발과 개념 중심 사고를 결합해 더 빠르고 자연스러운 학습을 실현합니다.

Vibe Learning이란?

Vibe Learning은 Vibe Coding 시리즈에서 성장한 방법론입니다. 50회의 AI 보조 개발 라이브를 거치면서 명확한 프레임워크가 형성되었습니다 — 의도와 멘탈 모델에 집중하고, 구현의 세부 사항은 AI에게 맡기며, 학습을 몰입 상태에 유지하는 것. 이 접근법은 이제 Catch Up AI의 모든 프로젝트에 적용됩니다.

핵심 원칙

  • 개념 우선 몰입 (Concept-First): 세부 구현에 들어가기 전, 고차원적인 논리를 먼저 파악합니다.
  • AI 가속 피드백: LLM을 24시간 개인 튜터로 활용하여 궁금증을 즉시 해결합니다.
  • 프로젝트 기반 마스터리: AI의 도움을 받아 실제 문제를 해결하는 프로젝트를 직접 구축하며 배웁니다.
  • 의도 중심 아키텍처: '무엇을 왜' 만들지에 집중하고, '어떻게'는 AI에게 맡깁니다.

"바이브 러닝의 목표는 당신을 AI 전문가로 만드는 것이 아니라, AI와 협업하는 전문가로 만드는 것입니다."

핵심 도구: VibeLearn AI

Vibe Learning 실험을 이끄는 핵심 도구는 VibeLearn AI입니다 — Vibe Coding 시리즈를 통해 직접 만들어진 맞춤형 학습 시스템입니다. 주제를 구조화하고, 연구를 안내하며, 각 실험의 산출물을 체계적으로 정리합니다.
